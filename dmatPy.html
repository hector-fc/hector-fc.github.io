Python para educadores

O projeto Python para educadores nasceu na necessidade de construir ferramentas computacionais para elaboração de material de ensino, em particular para professores e estudantes de Matemática. Também tem como objetivo auxiliar alunos de graduação e pós-graduação no processo de ensino e aprendizagem, pois o python é uma linguagem versátil e fácil de aprender, já que possui uma sintaxe simples e limpa, além disso, o uso de python vem crescendo, especialmente como linguagem de ensino de programação. Python possui bibliotecas consolidadas pela comunidade internacional, entre as quais está a biblioteca SymPy e Matplotlib, as quais são gratuitas e serão apresentadas por meio de resolução de problemas. O conhecimento desta linguagem também facilitará a aprendizagem de outras disciplinas ministradas no decorrer do curso.
