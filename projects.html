Packrec

Packrec: is a Julia package designed to solve 2D packing rectangles, where the goal is to maximize the number of rectangles that can be packed within a fixed container.

Packpoly

This project addresses the problem of packing identical regular polyhedra in arbitrary Euclidean space. The goal is to maximize the number of polyhedra that can fit within a convex set, modeling the problem as a nonlinear differentiable programming problem.
